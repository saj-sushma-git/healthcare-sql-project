# Healthcare SQL Project

## Project Overview
This project demonstrates SQL concepts using a healthcare dataset.

## Tables
- Patients
- Doctors
- Appointments

## Queries
1. Total appointments per doctor
2. Patients with more than one appointment
3. Patient and Doctor mapping
4. Doctors with no appointments
5. Total appointments per patients
6. Most visited doctor
7. Total appointments by city
8. Completed appointments per doctor
9. Highest completed appointments (RANK)
10. Doctors with completed appointments
11. Unique cities
12. Completed vs Pending appointments
13. Doctors with no appointments
14. Patients with more than one appointment
15. Combine Patient and doctor names
16. Patients from cities having appointments

## Conclusion
Covers joins, aggregation, subqueries, and window functions.
